How to use this tool?

Docker Image Builder is a helper tool to allow you to easily build Job Manager Client Docker images, and pack your own jobs into it.

You simply need to put your Job code in some python file, targz'd, zipped or not.

Then, you can build your Docker image through web user interface on the upload file package page or the Code Editor page

You can also script things using the API located on /build and access it using your favorite tool, like curl : 

curl -X POST http://0.0.0.0:5001/build \
  -H 'accept: application/json' \
  -H 'content-type: multipart/form-data' \
  -F 'name=test_upload' \
  -F 'tags=latest v1' \
  -F 'pip=tbx requests' \
  -F 'imports=test_upload' \
  -F 'apt=' \
  -F package=@./doc/examples/test_upload.py
